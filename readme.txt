Ni Sales Commission For WooCommerce provides the option to give the commission on sales orders to your sales agent. 

== Description ==
<p>
Sales Commission plug-in provides the option to give the sales commission to your sale agent or dealer based on the order total. Sales Commission plug-in provides the option to set commission based on the user role.  
</p>

<p>
Order report shows the commission on each order, order commission report gives the details of commission like commission mode, commission amount, order commission and order details.
</p>
<p>Commission reports show total commission earned by each user, you can filter the report by order periods.</p>
<h3>Key Feature of Ni WooCommerce Sales Commission</h3>
<ul>
	<li>Provide the option to set the commission base on the user role</li>
	<li>Set the commission in fixed and percentage mode </li>
	<li>Order report </li>
	<li>Commission report</li>
	<li>Option to set user Commission base on user role</li>
	<li>Filter report by order period</li>
	<li>Ajax Search</li>
</ul>

<h3>How Ni Sales Commission For WooCommerce works</h3>
<ul>
	<li>Install and activate the plugin</li>
	<li>In menu click on "user role" </li>
	<li>Set the commission mode (Fixed or Percentage) and commission to user role</li>
	<li>Click on save button</li>
	<li>make order</li>
	<li>In menu click on "commision report" for commision </li>
	<li>In menu click on "order report" for order </li>
</ul>
